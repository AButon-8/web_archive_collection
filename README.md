# Web Archive Collection



### Описание проекта
Репозиторий создан в рамках курса «Цифровая архивация современности. Веб-архивы, большие данные и их применение в исследовательской работе».
2 курс магистратуры НИУ ВШЭ «Цифровые технологии в гуманитарных исследованиях».

Проект представляет собой коллекцию веб-архивов сайтов, созданных с использованием инструмента [`wpull`](https://wpull.readthedocs.io/en/master/). Хранилище содержит метаданные WARC-файлов, полученные с помощью [`metawarc`](https://github.com/datacoon/metawarc) и результаты анализа коллекции с помощью ресурса [ArchiveReady](https://archiveready.com/) по метрикам [`CLEAR`](http://purl.pt/24107/1/iPres2013_PDF/CLEAR%20a%20credible%20method%20to%20evaluate%20website%20archivability.pdf).



### Сайты в коллекции:

Для веб-архивации были использованы материалы с трёх ресурсов, использовались разные подходы к их сохранению и обработке:

| Сайт                                                                            | Метод сбора      | Примечания                            |
| ------------------------------------------------------------------------------- | ---------------- | ------------------------------------- |
| [iskra-research.org](https://iskra-research.org)                                | ресурс полностью | wpull → WARC. Ресурс содержит материалы, связанные с революционной нелегальной газетой "Искра" |
| [marxists.org/lenin](https://www.marxists.org/russkij/lenin/)                   | скачивание с фильтрацией по разделу | Раздел сайта marxists.org, посвященный трудам В.И.Ленина, содержит избранные произведения теоретика марксизма   |
| [royallib.com/krupskaya_nadegda](https://royallib.com/author/krupskaya_nadegda) | скачивание через прямые ссылки | Страница с текстами Надежды Константиновны Крупской  в электронной библиотеке royallib.com |



### Методы и технологии сбора:
1. Прямое скачивание HTML страниц и связанных файлов с помощью `wpull`, результаты сохранены в формате WARC.
Сохраняются все страницы, медиа-файлы и структура сайта. Архив позволяет полностью восстановить исходный сайт локально. 
Удобно для анализа сайта как целого, с сохранением контекста публикаций, навигации и медиа-материалов. 

2. Рекурсивное скачивание с фильтрацией по разделу и сохранением страниц в WARC-файл. 
Позволяет собрать с сайта весь корпус текстов одного автора, включая HTML-страницы, не скачивая лишние разделы. 
Структура сохраняется в виде WARC и отдельных HTML файлов. Подходит для исследовательского анализа текстов и визуализации сетевых связей между страницами.

3. Скачивание готовых архивов .zip с текстами в формате .txt через прямые ссылки. 
Тексты представлены в читаемом виде, без HTML-разметки, каждый файл соответствует отдельному произведению. 
Это позволяет работать с текстами напрямую, без дополнительной очистки. Удобно для лингвистического и стилеметрического анализа текстов, подготовки корпусных данных.



### Структура хранилища
Описание каждого сайта и его WARC-архива находятся в одноименных папках.

```plaintext
web_archive_collection/
├── iskra-research/                 # Материалы сайта iskra-research.com
│   ├── metadata/.                  # Метаданные веб-архвива iskra-research
|   └── README.md                   # Анализ архивируемости и метаданных веб-архива
├── marxists.org-lenin/             # Материалы сайта marxists.org-lening
│   ├── metadata/.                  # Метаданные веб-архвива marxists.org-lenin
|   └── README.md                   # Анализ архивируемости и метаданных веб-архива
├── royallib.com-krupskaya_nadegda/ # Материалы сайта royallib.com-krupskaya
│   ├── metadata/.                  # Метаданные веб-архвива royallib.com-krupskaya_nadegda
|   └── README.md                   # Анализ архивируемости и метаданных веб-архива
├── README.md                       # Описание проекта и метрики
└── LICENSE                         # Лицензия
```
